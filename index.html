Reviewing the Wallboard Displays

Use these colorful, collapsible cards to quickly understand team capacity and work performance.

Set your filters first, then open the sections below. Each one explains what the chart/widget means and how to act on it.

Work Item Status
  • Waiting: Case is in backlog, not yet assigned.
  • Assigned: Routed and accepted by an agent but not yet being worked.
  • In Progress: Agent is actively working the case.
Agent Capacity

This chart shows:

  • Available Capacity: Remaining space to take new work.
  • Used Capacity: How much is currently being consumed.
Agent Capacity Status
  • At Capacity: Fully occupied.
  • Available: Ready to take new work.
  • Busy: Logged in but not accepting work.
  • Idle: Online but handling no work.
  • Offline: Not logged in.
Agent Presence Statuses

Shows how agents are spending their time (e.g., Available, In Meeting, On Lunch).

Wait Time

Tracks how long unassigned work waits in backlog before being picked up.
